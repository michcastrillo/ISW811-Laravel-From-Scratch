# Blade The Absolute Basics
Blade es el motor de plantillas de Laravel para las vistas. Es como una capa sobre PHP para hacer que la sintaxis requerida sea más limpia y concisa. 
- Archivo blade.php
    ```php
    @if
        //sintaxis
    @endif
    ```
- Archivo .php 
    ```php
    <?php
            if () {
                //sintaxis
            }
        ?>
    ```
